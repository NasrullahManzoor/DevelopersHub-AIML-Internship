# AI/ML Engineering Internship - DevelopersHub Corporation 

This repository contains the projects and tasks completed during **Phase-1** of the AI/ML Engineering Internship at **DevelopersHub Corporation**.

##  Projects Overview:

###  Stock Price Prediction 
- **Goal:** Predict future stock prices using historical data.
- **Model:** Linear Regression.
- **Tech:** Python, Pandas, Scikit-Learn, Yahoo Finance.

###  House Price Prediction 
- **Goal:** Predict property prices based on features like size and location.
- **Model:** Gradient Boosting Regressor.
- **Tech:** Python, Scikit-Learn, Matplotlib.

###  Iris Flower Classification 
- **Goal:** Classify iris flowers into species based on measurements.
- **Tech:** Exploratory Data Analysis (EDA), Data Visualization.

###  Bonus Task: Advanced Iris Analysis
- **Description:** An advanced version of the Iris project featuring in-depth EDA, correlation heatmaps, and optimized model performance.

---
**Tools Used:** Python, Jupyter Notebook, NumPy, Pandas, Seaborn, Matplotlib.
